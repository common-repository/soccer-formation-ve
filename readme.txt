== Description ==
This plugin generates vertical soccer formations with dynamic SVG images.

### Upgrade to Soccer Engine

If you have a soccer-related website, consider purchasing [Soccer Engine](https://daext.com/soccer-engine/). A complete solution for displaying soccer data in soccer club's websites, soccer news blogs, transfer market blogs, and more.

### Soccer teams

Store jersey numbers and names of the players for an unlimited number of teams with the **Formations** menu.

### Drag and drop editor of soccer formations

Use the drag and drop editor available in the **Layouts** menu to create custom formations.

This editor supports a custom number of players per formation. You can, for example, create 11 vs 11 formations, 7 vs 7 formations, 6 vs 6 formations, and more.

### Customizable elements

In terms of customizability, the plugin comes with the following options:

* Player Number Color
* Player Name Color
* Player Number Background Color
* Player Name Background Color
* Field Background Color
* Field Lines Color
* Font Size
* Font Weight
* Font Family
* Load Google Font
* Field Top Margin
* Field Bottom Margin

### Vector Graphics

No matter what screen resolution or size are being at, the soccer formations generated by this plugin scale to any dimension without losing quality.

### Responsive

The size of the soccer formations generated by this plugin is automatically adapted based on the container.

== Installation ==
= Installation (Single Site) =

With this procedure you will be able to install the Autolinks Manager plugin on your WordPress website:

1. Visit the **Plugins -> Add New** menu
2. Click on the **Upload Plugin** button and select the zip file you just downloaded
3. Click on **Install Now**
4. Click on **Activate Plugin**

= Installation (Multisite) =

This plugin supports both a **Network Activation** (the plugin will be activated on all the sites of your WordPress Network) and a **Single Site Activation** in a **WordPress Network** environment (your plugin will be activated on a single site of the network).

With this procedure you will be able to perform a **Network Activation**:

1. Visit the **Plugins -> Add New** menu
2. Click on the **Upload Plugin** button and select the zip file you just downloaded
3. Click on **Install Now**
4. Click on **Network Activate**

With this procedure you will be able to perform a **Single Site Activation** in a **WordPress Network** environment:

1. Visit the specific site of the **WordPress Network** where you want to install the plugin
2. Visit the **Plugins** menu
3. Click on the **Activate** button (just below the name of the plugin)
